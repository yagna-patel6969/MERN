## How to Run Locally

### Option 1: Open directly in browser
1. Clone the repo:
   ```bash
   git clone https://github.com/yagna-patel6969/MERN.git
   cd MERN
   ```
2. Open any project’s main `.html` file in your browser.

### Option 2: VS Code Live Server (recommended)
1. Open the repository in VS Code
2. Install the **Live Server** extension
3. Right-click an entry `.html` file → **Open with Live Server**
